# Exam 01: Grundlagen

## Anweisungen

1. Klone dieses Repository
2. Gehe jede Aufgabe nacheinander durch, jede Aufgabe befindet sich in einem separaten Ordner
3. Jede Aufgabe hat eine `README.md` Datei mit Anweisungen
4. Vervollständige die Aufgabe und pushe deinen Code in dieses Repository

Auch wenn du eine Aufgabe nicht abschließen kannst, bitte pushe deine Arbeit. Wenn du bei einer Aufgabe nicht weiterkommst, gehe zur nächsten über und komme später darauf zurück - nutze deine verfügbare Zeit effektiv.

Denke daran, deinen Code regelmäßig zu commiten (auf das richtige Verzeichnis achten!) und zu pushen. Stelle sicher, dass du vor der Deadline fertig bist, nach der Pull Requests nicht mehr akzeptiert werden.

## Regeln

- Teile deinen Code nicht mit anderen
- Diskutiere die Fragen nicht mit deinen Klassenkameraden
- Verwende keine KI-Tools, um Code für dich zu generieren
- Du *darfst* Fragen an AI-Chatbots stellen
- Du *darfst* das Internet nutzen, um nach Antworten zu suchen, solange du nicht wörtlich kopierst
- Du *darfst* deine eigenen Notizen und die der Klasse verwenden

## Bewertung

- Die Bewertung erfolgt auf einer Skala von 0 bis 100
- Du erhältst eine Gesamtpunktzahl, aber keine Einzelpunktzahlen für die Aufgaben
