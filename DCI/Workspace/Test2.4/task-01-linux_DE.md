# Aufgabe 01 - Linux

Füge deine Antworten unter den Fragen in dieser Datei hinzu. Zum Beispiel:

```
- Frage 0: Welcher Befehl wird verwendet, um eine leere Datei im Terminal zu erstellen?
- Antwort: `touch filename.txt`
```

Wenn du dir bei einer Antwort nicht sicher bist, schreibe, was du über das Thema weißt oder was du versucht hast, um es zu lösen.

## Fragen

- Frage 1: Welcher Befehl wird verwendet, um den Inhalt eines Verzeichnisses aufzulisten, einschließlich versteckter Dateien und Ordner?

- Antwort: ls la

- Frage 2: Welcher Befehl wird verwendet, um das aktuelle Arbeitsverzeichnis anzuzeigen?

- Antwort: pwd

- Frage 3: Welcher Befehl wird verwendet, um ein neues Verzeichnis zu erstellen?

- Antwort: mkdir

- Frage 4: Nenne 3 verschiedene Wege, um Hilfe zu einem Befehl zu bekommen.

- Antwort: git --help, git -h, 

- Frage 5: Welcher Befehl wird verwendet, um eine Datei zu löschen?

- Antwort: rm -r für das Verzeichniss und rm für die Datei.

- Frage 6: Welcher Befehl wird verwendet, um in ein neues Arbeitsverzeichnis zu wechseln?

- Antwort: cd

- Frage 7: Welcher Befehl wird verwendet, um eine Liste der zuvor ausgeführten Befehle anzuzeigen?

- Antwort: history

- Frage 8: Was ist "Ubuntu" in Bezug auf Linux?

- Antwort: Ubuntu ist ein Betriebssystem, das auf Linux basiert.

- Frage 9: Was ist der Zweck des Verzeichnisses `/etc/`?

- Antwort: Es ist ein zentraler Ort, der wichtige Dateien und Verzeichnisse speichert.

- Frage 10: Wie würdest du das Programm `terminator` über die Kommandozeile installieren?

- Antwort: sudo apt install terminator

- Frage 11: Was ist der Zweck einer Umgebungsvariablen?

- Antwort: Eine Umgebungsvariable speichert wichtige Informationen, zum Beispiel den Pfad zu Dateien oder Einstellungen für Programme. 

- Frage 12: Du befindest dich im Verzeichnis `/home/user/projects/myproject`. Welchen Befehl würdest du verwenden, um in das Verzeichnis `/home/user/Downloads/` zu wechseln?

- Antwort: cd /home/user/Downloads/
